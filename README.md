# fizz-buzz-pytorch

Solve [Fizz Buzz][1] with [PyTorch][2].

Read [Joel Grus' blog post][3].

## Quickstart Guide

1. [Install poetry][4]

2. Install dependencies

   `poetry install`

3. Solve Fizz Buzz with PyTorch

   `poetry run python3 src/model.py`
